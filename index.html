About Me

My name is Martha Nakkazi. I am Front End Web Developer, with experience using HTML, CSS, SASS and JavaScript to develop beautiful and responsive websites. I have designed and developed a number of projects, a few of which are featured down below.

Portfolio

Mobile First Responsive Project

This project demonstrates my ability to design websites using a mobile first approach. I designed it in a mobile layout first (320px viewport), then used media queries to style the larger viewports (768px and 1024px). Flexbox was used to layout the main section of the project, and as the viewport gets smaller, we can see the sections getting stacked on top of each other to support smaller screens.

Web App Dashboard Project

The web app dashboard project demonstrates my ability to build a website in its entirety, that is, using HTML, CSS and JavaScript. This particular website offers the option of user interaction through throwing timed error messages when a user fails to fill in every section on the form, and a 'success' alert box that pops up when a message has been sent successfully. I used a callback on the setTime out function to achieve this.

Employee Directory Project

The ability to understand and use Fetch API is another skill I possess, as shown in this project. This one randomly generates 12 users from an API and displays a new set each time a user reloads the page. User interaction with the page is also enabled through the previous and next buttons on the modal window that appears when an employee is selected.

Game Show App Project

The backbone of this project is JavaScript. It relies on it from start - when the user clicks the 'Start Game' button, to finish - when the user either wins or loses and decides to give it another try, by clicking the 'Reset Game' button. I used DOM manipulation to select the elements to be affected by user clicks. Event listeners would determine what would follow the clicks through the callbacks defined within them. 'Math.random' was also used to select what phrase went on each turn.

Interactive Photo Gallery Project

The interactive photo gallery project uses BaguetteBox to display an image and its caption to a user when they select it. The bar atop also comes in handy when the user wants to look up a particular picture. It uses the 'includes()' method to filter out each image whose caption doesn't include the words being typed into the search bar. As for the CSS, I used Grid to layout the images and 'auto-fit' to avoid having to use media queries on different viewports.

Web Style Guide Project

The web style guide project showcases my ability to use SASS in styling websites. I used variables and mixins to avoid having to retype the same styles over and over again. Nesting was used for elements that share styles, and the partials enabled me to group similar styles together, hence creating even shorter, easier to manage stylesheets. I used 'sass-watch' in the command line to monitor any change made to my scss file, which would then later be updated in the final css file.
